MFC+opencv2.7+VS2012实现图像拼接

发表于 2017-02-02 | | 阅读次数

前言

opencv中有现成的sticher类，里面集成了所有关于图像拼接的函数，所以在这方面，我们完全不用自己实现。若仅仅想在MFC环境下完成图像拼接，那么看看这篇博客就能解决了，但是如果还想对拼接进行其他的操作，比如特征点的提取和显示等，则还需要自己去深入研究了。

实现的功能

  1. 对需要拼接图像的选择和显示。
  2. 对上述图像进行resize操作，使其能够进行拼接。
  3. 拼接图片并展示。
