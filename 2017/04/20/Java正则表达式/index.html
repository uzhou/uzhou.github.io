正则表达式

定义及基础概念

定义

正则表达式是一个用于匹配字符串的字符串模板，可以将其看成一个强大的字符串处理工具。

基础概念

String和正则表达式

由于正则表达式是用于匹配和查找字符串的工具，因此String类中也提供了多个方法用于支持正则表达式。比如：

boolean matches(String regex):判断该String对象与regex是否匹配。

String replaceAll(String regex,String replacement):将该String对象中所有与regex匹配的子串都用replacement替换掉。

正则表达式的形式

可以将正则表达式是一个特殊形式的字符串，就如同上述的String中的method一样，将正则表达式以一个特殊字符串的形式作为参数传到了方法中。所以构建正则表达式最重要的部分就是创建一个合乎我们需求的特殊字符串，对此Java中也给出了一些regex的规范。

Java中的两个正则相关类

Pattern

上面提到正则表达式是一个特殊的字符串，那么JVM是如何将这个特殊字符串和我们一般意义上的字符串区分开来用于字符串匹配的呢？这里就需要进行一个转换，把这个字符串转化成JVM能够理解的内存形式。JDK中有两个正则相关的类，一个是Pattern，另一个是Matcher。
Pattern是将正则表达式的字符串形式转化为Java的内存形式，并提供了一些简单的匹配方法。

上述代码中的str[0]是12381，str[1]是2321312。想要进行一些复杂的匹配操作或者描述匹配状态时，需要Matcger类。

Matcher

Matcher类提供三个匹配操作方法,三个方法均返回boolean类型,当匹配到时返回true,没匹配到则返回false。其中matches()对整个字符串进行匹配,只有整个字符串都匹配了才返回true；lookingAt()对前面的字符串进行匹配；find()只要字符串中有目标字符串的子串时就返回true。
另外，Matcher类还提供了几个描述匹配的状态信息，比如捕获分组的个数，捕获分组信息等。
