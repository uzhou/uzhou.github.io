Word Search

发表于 2017-03-06 | | 阅读次数

发现自己好长时间没有刷题了，惭愧！这种事落下一天就想落下第二天、第三天、、、，好在现在捡回来还不算太晚。
这一题是一个典型的递归问题，所以不算太难：

  1. 首先需要一个isMark的二维boolean数组，用于记录矩阵中的某个位置是否被遍历过；
  2. 确定递归过程中的起点状态、退出条件（包括超出边界，不满足题意等）、下一个递归的起点状态；
  3. 然后就是对每一个位置进行递归操作。
    上面说的也是典型的递归问题解决方法，一般的递归问题按照这种套路很容易就能解决。
