Unique Binary Search Trees

发表于 2017-03-13 | | 阅读次数

第一想法是暴力法，枚举所有可能的序列，对此序列进行判断，看是否是合法的二叉搜索树，但后来感觉应该没有这么复杂。因为1~n的二叉搜索树个数等于k ~n+k的个数,所以一旦确定一个元素为根节点（假设该节点值为k，总节点数为n），那么其左边一定是小于该根节点元素的连续序列组成的二叉搜索树，个数为F[k-1],F[k-1]表示当总节点数为k-1时二叉搜索树的总个数；同理右边的个数为F[n-k]，总个数为两者相乘。
