Unique Binary Search Trees2

发表于 2017-03-14 | | 阅读次数

借助Unique Binary Search Trees的思路，按根节点的值分类。由于无法直接获取节点个数为n时的所有树，我们需要借助1~n-1的结果才能得到最终答案。所以第一个循环表示的是当前节点个数从2到n；第二个循环表示根节点的情况是1~n；第三、第四个循环则表示左边树的情况和右边子树的情况，这时候要注意右边子树的值需要加上一个增值（这个值等于当前根节点的值）。
