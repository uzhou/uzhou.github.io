Search in Rotated Sorted Array

发表于 2017-03-07 | | 阅读次数

在一个反转的递增数组中进行查找的问题，如果是正常的递增序列，那么时间复杂度最快的是二分查找。很明显本题需要对二分查找略做修改，由于不存在重复元素，因此以该序列的中间位置mid作为分界。要么beginIndex到mid位置的元素序列为递增序列，要么从mid到endIndex位置的元素为递增序列。因此只要target在这个递增序列中，就可以进行二分查找，如果不在，那么说明在另一个序列中，此时只需要对另一个序列进行递归操作。
