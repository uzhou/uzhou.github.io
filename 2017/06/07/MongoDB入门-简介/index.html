MongoDB入门--简介

发表于 2017-06-07 | | 阅读次数

概述

MongoDB 是一个基于分布式文件存储的数据库。由 C++ 语言编写。旨在为 WEB 应用提供可扩展的高性能数据存储解决方案。
MongoDB 是一个介于关系数据库和非关系数据库之间的产品，是非关系数据库当中功能最丰富，最像关系数据库的。

关系和非关系型数据库

既然MongoDB是非关系型数据，那么它和我们平时常用的关系型数据库比如MySQL有什么区别呢。
首先，这个“关系”代表的是什么？我们知道MySQL中对数据管理是通过表结构来管理的，一个表就代表了一个实体的集合，而关系就是实体与实体之间的关系（最常见的关系有1对1,1对多，多对多等）。
另外，菲关系型数据库遵循最终一致性原则，而不是关系型数据库的ACID原则。

元数据

元数据的定义比较简单，描述数据的数据。在MongoDB中，元数据指的是描述数据库信息的数据，这些数据都存放在system这个特殊集合中。

对MongoDB的第一印象

第一次接触MongoDB，最大的印象就是MongoDB中的数据存储是以BSON形式存储的，并且同一个集合中的不同的文档（相当于同一个table中不同的行）可以有不同的结构，但是一般文档之间的结构基本一致，这也许就是为什么说MongoDB是最像关系型数据库的非关系型数据库吧。
