DivideTwoIntegers

发表于 2017-01-09 | | 阅读次数

不能使用乘除和取余运算，来完成一个除法。想了半天，想了一个naive级的算法，就是被除数除数都取绝对值，然后每次减一个除数，商随之加1，直到被除数小于除数为止，正负号另判。但是十分遗憾的是超时，没办法，每次不能只减一个除数，我就每次减100个除数，不足时，再减一个除数。这样就解决了超时问题，但是执行效果令人汗颜，只超过了0.19%的用户，好歹试过了，就不纠结了。
