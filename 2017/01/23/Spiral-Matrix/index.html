Spiral Matrix

发表于 2017-01-23 | | 阅读次数

螺旋矩阵(m n)问题，如果是(n n)的矩阵那么十分简单。由于不是方阵，所以要考虑m和n的大小。要考虑的因素有：循环次数(受min(m,n)影响)；最后一个循环处理(受m,n大小影响)。一次性解决这个问题还是有点难度的，只能通过不断的提交找出自己的算法的不足，然后慢慢的修改就可以了。
