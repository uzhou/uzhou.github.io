Multiply Strings

发表于 2017-01-14 | | 阅读次数

标准的大数相乘算法，主要是解决一些大的整数相乘会溢出的现象。这一题的思路，和我们平时做乘法计算是一样的。把这个乘法看成（n-1）次加法，不过每一次加法的时候都要对加数进行处理一下，即乘上10^(i)再和被加数相加。注意，每一次加数的获取的最后要判断carry是否为0。
