Generate Parentheses

发表于 2017-01-02 | | 阅读次数

这是一个很明显的递归问题，对于递归，最关键的是找出相同点和边界，这一题要求我们生成n对括号并且是匹配的。那么就有2n个空位让我们去填”(“或者”)”，如何选择”(“或者”)”。其实从第1个位置到第2n个位置，都满足一个条件，那就是剩余的左括号<=右括号否则就是不匹配的。只要抓住这个共同点，以及知道递归的边界，那么就很容易解决这个问题了。
