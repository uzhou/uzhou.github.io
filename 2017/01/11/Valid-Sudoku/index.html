Valid Sudoku

发表于 2017-01-11 | | 阅读次数

思想还是遍历，根据数独的规则，要进行三大类的判断：1）判断行是否重复 2）判断列是否重复 3）判断每一个小的九宫格是否重复；分类之后发现最难处理的是第3类情况。难点在于如何将从99的方格内取出对应的小九宫格，经过观察和计算得第i个九宫格元素对应于board[j+i/33][k+i%3*3],其中j表示小九宫格的第j行，k表示小九宫格的第k列。取出对应的九宫格又可以转成第1、2类问题乐。效率为2.17%,确实不忍直视。
