Windows10下caffe+cifar10+ipython物体分类的可视化(一)

发表于 2017-01-28 | | 阅读次数

前言

本来只是想做一个图像拼接作为cv作业的，但是发现寒假时间比较充足，于是就试着能不能作物体分类。本程序是在windows10+caffe+cifar10+python+VS2013 community的环境下调试通过的，并用ipython notebook实现了可视化，对一些中间结果进行了显示。PS：windows下caffe的配置及其恶心，出现的问题让你不知所措，如果有条件可以试着用linux。

准备工作

  • VS2013 community的下载和安装。
  • CMake安装（一个跨平台的构建工具，能够生成不同平台的目标文件，如vs的.vcproj）
  • 安装python并配置环境变量（最好安装Anaconda，里面集成了很多python的组件，对于后面的ipython notebook也能够支持）
