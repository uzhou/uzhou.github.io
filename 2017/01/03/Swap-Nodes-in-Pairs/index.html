Swap Nodes in Pairs

发表于 2017-01-03 | | 阅读次数

典型的单链表问题，最简单的应该是在遍历的时候直接改变相邻的节点的值，但是这种做法被排除了。我们只能通过单链表的性质来解决，借鉴C/C++中的指针思想，改变节点顺序可以解决该问题。具体的思想看一个图就可以了：

如上图所示，我们需要用到两个游标，一个front指向需要交换的第一个节点的前一个节点，由于题目给的head代表的是整个链表的第一个节点而不是头节点，所以我们需要做一下简单的处理，使一开始的front指向一个值为-1的节点，并且next指向head，还需要一个游标rear指向需要交换的第二个节点的下一个节点。所以我们看到的一开始的链表状态就是1-1，1-2到1-4就是交换的流程图了，这个应该大家都能看懂。最后，我们交换当前两个节点，需要将front向后移动两个单位。
注意，我们返回的不总是head，当链表长度>=2时是原链表的第二个节点，而当链表长度小于2时返回的才是head。
