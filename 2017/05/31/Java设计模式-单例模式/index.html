Java设计模式--单例模式

发表于 2017-05-31 | | 阅读次数

概述

单例模式表示某个类只能有一个实例化的对象，且为该对象提供一个全局访问点。
要实现单例模式必须有下面三个要点：

  1. 构造方法必须私有
  2. 要有一个私有的静态引用
  3. 因为构造方法私有，所以产生该单例对象的方法必须是static，该方法将为2中的私有引用赋值，如果已赋值则直接返回该引用。

单例实现

方法一

此方法称为饿汉式，线程安全。

方法二

饱汉式，非线程安全，因此多线程下会出现多个实例对象。

方法三

线程安全，但是代价大，每次都要进行同步。

方法四

最佳实现方式，仅仅第一次进行了同步处理，效率、内存占用都是最佳。
