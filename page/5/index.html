Restore IP Addresses

发表于 2017-03-11 | | 阅读次数

这一题比较简单的算法就是穷举法，不过在穷举之前要做些数据处理。由于IP地址的每一个区间内的整数都是0-255的合法数（所谓合法是指不会出现前置0）。所以我们可以将以每一个数字开始的合法数存到对应该位置的List中，然后将所有的list存放到map中，之后需要穷举的只是这些list内的String。穷举采用递归法，要注意递归的边界，起点，中间状态等必要因素。

Partition List

发表于 2017-03-10 | | 阅读次数

分割链表操作，难点是保持原有元素的相对顺序不改变。如果另外分配空间的话十分简单，只需要重新建立一个链表就ok了。但是还是要想想有没有o（1）空间复杂度的算法。经过观察知道：大于等于val的数（有可能是连续的也有可能不是连续的）可以与后面的小于val的数交换。这样最后得到的链表就是要求的链表，时间复杂度为o（n）。

Merge Sorted Array

发表于 2017-03-08 | | 阅读次数

将两个递增的序列a、b合并到递增序列a中，因为给出a的大小是满足m+n的所以不必考虑空间的重新分配。这种合并会出现两种情况，一种是a的最大值大于b的最大值（这种情况不需要做任何处理）；另一种是b的最大值大于a的最大值（需要将b剩下的元素直接合并到a中）。利用双指针分别遍历a、b，当b中的当前元素比a中的当前元素要小的话，就将a后移一位，并将b的当前元素插入进来。
