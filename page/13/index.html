Hexo+Git+Node.js 搭建自己的github博客

发表于 2016-11-25 | | 阅读次数

写在之前的话

本人第一次搭建github博客个人主页，着实踩了不少坑，为了让后来者少踩坑，就在第一篇博客中讲讲如何搭建的问题。

一、环境搭建

1、1 Git安装

先从Git官网上下载Git安装包：https://git-scm.com/downloads。
点击安装，一键next的安装过程，没有什么需要注意的。

1、2 Node.js安装

下载地址：https://nodejs.org/en/
要根据自己的平台选择合适的版本一键安装。

二、github相关设置

2、1 申请github账号

去github官网申请一个账号，注意要进行邮箱验证，并记住邮箱和用户名，在下面的设置中会需要到。

2、2 SSHkey的设置

1、由于在步骤一中完成了环境搭建，所以我们可以在任意位置右击，选择git bash here。
在出现的窗口先设置github的用户名和邮箱：
$ git config –global user.name “uzhou”
$ git config –global user.email “uzhou@zju.edu.cn”

2、然后生成sshkey

这个时候我们需要输入$ ssh-keygen -t rsa -C “uzhou@zju.edu.cn”
邮箱是申请github账号的邮箱。按3个回车，密码为空。

Your identification has been saved in /home/tekkub/.ssh/id_rsa.
Your public key has been saved in /home/tekkub/.ssh/id_rsa.pub.
The key fingerprint is:
XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

出现上述内容，说明生成成功。生成了两个文件夹一个id_rsa还有一个id_rsa.pub。

3、将sshkey添加到github上：
输入指令：ssh-add ~/.ssh/rsa

执行到这一步时，我出现了一个cannot open connection的提醒，这时候添加eval ssh-agent -s 后解决了该问题。成功后会显示Identity added: /Users/xxx/.ssh/id_rsa (/Users/xxx/.ssh/id_rsa)

4、添加sshkey到github上
复制id_rsa.pub中的所有内容，进入自己的github上，找到settings中的SSHKEY选项，选择new SSH
并将内容复制好，title可以任意填写不影响。

5、输入ssh git@github.com成功时会显示

Hi your_name! You’ve successfully authenticated, but GitHub does not provide shell access.

Connection to github.com closed.
一般在此之前会有个提示，直接输入yes即可。

三、正式安装

3、1 hexo安装

1、首先创建一个hexo文件夹，我创建在了D盘下。

2、执行如下命令：

npm install -g hexo

安装过程中遇到的warn可以直接忽略。

3、点开hexo文件夹，右击选择git bash here

输入命令：hexo init

4、安装依赖包

输入命令：npm install

5、本地查看

首先要生成一下：hexo g
然后部署一下：hexo s
在浏览器中输入localhost:4000查看效果

6、部署到github上

在hexo文件夹下的_config.yml文件中修改如下信息

deploy:
type: git
repository: https://github.com/uzhou/uzhou.github.io.git
branch: master

完成上述操作后，只需要重新生成和部署就可以了：

hexo -g

hexo -d

这样就能通过uzhou.github.io来访问个人主页。

四、主题修改

可能有些小伙伴并不喜欢原来的主题，这个时候我们需要自己更换一下主题，我推荐next简约型主题。

1、从github上clone下来theme-next：
在hexo目录下右键git bash通过命令
git clone https://github.com/iissnan/hexo-theme-next themes/next将主题复制到themes目录下。

2、修改hexo目录下的config文件，将theme: landscape改为theme: hexo-theme-next

3、重新hexo g hexo d就可以看到新主题了。

其他配置，例如头像、语言等可以参考官方文档。
